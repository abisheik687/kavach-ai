## Overview
Binary deepfake detection dataset for fine-tuning KAVACH-AI ensemble models.
Faces extracted from source datasets using MTCNN with 5-point landmark alignment.

## Label Schema
| Label | Value | Meaning |
|-------|-------|---------|
| Real  |   0   | Genuine unmanipulated face |
| Fake  |   1   | Deepfake / AI-generated / face-swapped |

## Target Composition
| Split | Real | Fake | Total |
|-------|------|------|-------|
| Train | 20,000 | 20,000 | 40,000 |
| Val   |  2,500 |  2,500 |  5,000 |
| Test  |  2,500 |  2,500 |  5,000 |
| **Total** | **25,000** | **25,000** | **50,000** |

## Data Sources
| Source | Type | Licence | Split usage |
|--------|------|---------|-------------|
| FaceForensics++ (c23) | Video deepfakes | Research only | Train+Val |
| Celeb-DF v2 | High-quality celeb deepfakes | Research only | Train+Val |
| DFDC Preview | Diverse demographics | Apache 2.0 | Train |
| WildDeepfake | In-the-wild | Research only | Train+Test |
| Synthetic (bootstrap) | Generated by synthetic_gen.py | MIT | Dev/test only |

## Preprocessing Pipeline
1. MTCNN face detection (confidence threshold: 0.90)
2. 5-point landmark affine alignment (eyes horizontal)
3. Crop + resize to 224×224 RGB PNG
4. Hash-based deduplication
5. Low-resolution filter (reject < 112px in either dimension)
6. Stratified train/val/test split by (label, source_dataset)

## Augmentation (training split only)
| Transform | Parameters |
|-----------|------------|
| RandomHorizontalFlip | p=0.5 |
| RandomRotation | ±10 degrees |
| ColorJitter | brightness=0.2, contrast=0.2, saturation=0.1 |
| JPEGCompression | quality 50–95 (simulate social-media recompression) |
| GaussianBlur | sigma 0.1–2.0, p=0.15 |
| RandomPerspective | distortion=0.1, p=0.2 |
| Normalise | ImageNet mean/std |

## Known Limitations
- FF++ videos are mostly Western/Caucasian faces — demographic bias present
- Synthetic bootstrap samples do NOT represent real GAN failure modes
- No audio deepfake samples included (separate pipeline required)
- Test set leakage risk if same source video contributes to train and test
  (mitigated by source-stratified splitting but not fully eliminated)

## How to Reproduce
```bash
# 1. Extract faces
python training/data_pipeline.py --input_dir data/raw/FaceForensics/fake \
  --output_dir data/processed/fake --label 1
python training/data_pipeline.py --input_dir data/raw/FaceForensics/real \
  --output_dir data/processed/real --label 0

# 2. Build manifest
python training/build_manifest.py \
  --real_dir data/processed/real \
  --fake_dir data/processed/fake \
  --output   training/dataset_manifest.csv

# 3. Smoke-test loader
python training/dataset.py training/dataset_manifest.csv
```
